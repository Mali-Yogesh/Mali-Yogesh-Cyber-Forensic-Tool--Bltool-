# Mali-Yogesh-Cyber-Forensic-Tool--Bltool-
A forensic-grade BitLocker decryption tool for Windows disk images. Supports recovery key &amp; TPM-based unlock, AES-XTS/CBC decryption, NTFS parsing, file extraction, disk imaging, file carving, and virtual mounting. Built with C++ and OpenSSL. Ideal for data recovery, forensic investigation, and BitLocker research.
